### ② 搜索增强 ✅

#### A. 拼音搜索

**功能**：
- 输入拼音可匹配诗词标题和内容
- 支持全拼（如 `jingyesi` 匹配「静夜思」）
- 支持首字母（如 `jys` 匹配「静夜思」）
- 自动缓存拼音计算结果，避免重复计算

**技术实现**：
- 库：`pinyin-pro`（支持多音字、声调、首字母）
- Hook：`src/hooks/useSearch.ts`（封装拼音匹配逻辑）
- 工具：`src/lib/pinyin.ts`（`matchWithPinyin` / `pinyinMatchScore`）

**示例**：
- 搜索 `chun` → 匹配所有标题含「春」的诗
- 搜索 `sysj` → 匹配「水调歌头·明月几时有」（首字母）

---

#### B. 语义搜索

**功能**：
- 用自然语言描述想找的诗词（如「思乡的诗」「写山的作品」）
- AI 理解意图后推荐相关诗词
- 结果包含推荐理由
- 支持中英文查询

**技术实现**：
- API：`/api/ai/search`（服务端）
- 数据源：直接读 Redis `poems:all`（避免客户端 API 调用）
- 缓存：7 天 TTL（仅站点默认 Key）
- 模型：复用 `AI_TEXT_MODEL`（如 `agnes-2.0-flash`）

**使用方式**：
1. 打开 `/search` 页面
2. 点击「开启 AI 语义搜索」按钮
3. 输入自然语言查询（如「关于月亮的诗」「写离别的」）
4. AI 返回相关诗词列表

**实测**：
```
POST /api/ai/search
{"query":"poems about moon","limit":5}

Response:
{
  "results": [
    {"id":"poem-xxx", "reason":"月下独酌，邀月共饮"},
    {"id":"poem-yyy", "reason":"明月几时有，把酒问青天"}
  ]
}
```

## 技术细节

### 拼音匹配算法
```ts
// 三级匹配优先级
1. 原文完全匹配 → 100 分
2. 原文前缀匹配 → 90 分
3. 原文包含匹配 → 80 分
4. 拼音完全匹配 → 70 分
5. 拼音前缀匹配 → 60 分
6. 拼音包含匹配 → 50 分
7. 首字母完全匹配 → 40 分
8. 首字母前缀匹配 → 30 分
```

### 语义搜索 Prompt
```
用户查询：{query}
诗词列表：[{id, title, author, content, tags}, ...]

要求：
1. 理解用户查询意图（主题、情感、意象、风格）
2. 选出最相关的诗词（最多 10 首）
3. 返回 JSON：{"results":[{"id":"...", "reason":"简短理由"}]}
```
