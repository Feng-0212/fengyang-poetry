### 2. 每日一诗（首页新功能）
- 组件: `src/components/collections/PoemOfTheDay.tsx`
- 逻辑:
  - 按日期确定性选诗（`seed = YYYYMMDD`），所有访客同一天看到同一首
  - 优先当季诗词，无则全库随机
  - 展示：节气标签 + 日期、诗题、作者·朝代、前两句预览、「品读全诗」入口
- 样式：与现有卡片体系一致（水墨晕染、季节色边框、Mashan 字体）
- 位置：首页 HeroSection 下方、CollectionGrid 上方

### 3. 性能优化
- **Google Fonts**: 从 CSS `@import` 移至 `<head>` `<link>`，避免 CSS 解析阻塞，字体请求可并行
- **图片懒加载**: AI 配图预览、壁纸预览添加 `loading="lazy" decoding="async"`

### 4. 数据备份（每日快照）
- 路由: `/api/backup`（GET，需 `?key=<CRON_SECRET>` 或 `Authorization: Bearer`）
- 机制:
  - 每日 03:00 北京（19:00 UTC）自动快照 `poems:all` + `collections:all`
  - 键名: `backup:poems:YYYY-MM-DD`、`backup:collections:YYYY-MM-DD`
  - TTL: 30 天；索引维护 `backup:index`（保留最近 60 条）
- 安全: `CRON_SECRET` 环境变量保护；本地/未配置时不拦截（方便开发）
- 说明: 这是**同库快照**，防"手滑删数据"；异地容灾（GitHub/对象存储）需后续另配
