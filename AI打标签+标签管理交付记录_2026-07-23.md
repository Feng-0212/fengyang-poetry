## 目标
两个功能并行交付：
1. **AI 自动打标签** — 在 AI 面板一键生成 2–3 个标签建议，可选后保存
2. **标签管理** — 设置页查看/重命名/合并/删除标签（需密码保护）

---

## 已完成

### 1. `POST /api/ai/tags` 路由
- 复用 `resolveConfig`（支持个人 Key / 站点默认）
- 提示词设计：从预定义标签池（思乡/山水/豪放/婉约/田园/边塞/咏物/送别/怀古/闺怨/禅意/爱情/羁旅/隐逸/闲适/离愁/壮志/悲秋）+ 模型自由补充中选 2–3 个
- 输出格式：纯 JSON `{"tags":["标签1","标签2"]}`，含 fallback 解析逻辑
- 关闭思考机制（`enable_thinking: false`）

### 2. `src/lib/ai.ts` — `getAiTags()` 客户端函数
```typescript
export async function getAiTags(
  title: string, content: string, author?: string
): Promise<string[]>
```

### 3. `AiPanel.tsx` 集成
- 新增「🏷️ AI 打标签」按钮（赏析/配图按钮旁边）
- 点击后调 AI → 显示标签建议面板（每个标签可取消勾选）
- 保存：合并到现有标签（去重，最多 8 个），调用 `onUpdated` 刷新父组件

### 4. `TagManager.tsx` 组件
- 加载时从 API 拉取全部诗词，统计各标签出现次数，按频次降序显示
- 三种操作（均需密码）：
  - **重命名**：输入新标签名，批量替换所有诗词
  - **合并**：选目标标签，将来源标签的诗词合并进去（目标已有时跳过）
  - **删除**：从所有诗词中移除该标签
- 操作完成后显示成功/失败消息，`key` 变化触发自动刷新列表

### 5. 设置页接入
- 插入「标签管理」section（数据导入之前）
- 调用 `TagManager`，`onDone` 用 `key` 增量触发刷新

## 技术细节
- `mergeTarget` 用 `useState<string>`（非 `Record<string,string>`），避免 TypeScript 报错
- 合并操作：目标标签已存在时跳过，避免同诗重复添加
- 标签上限 8 个（与写诗表单约束一致）
